Scratch 3 Converter (Alpha)

Hey everybody! I don't use Scratch anymore and wasn't really planning to make this. But after the success of my Scratch 2 converter and a lot of emails and comments, I figured I'd give it a try after all. Enjoy!


About

Unlike the previous version, Scratch 3.0 is entirely web-based (made with Node, webpack, and Electron) so I had to completely rewrite this from the ground up. Rather than a SWF file, this program produces Electron-based native apps for macOS, Windows, and Linux. The converter itself is written in Java.


Download

Be warned, this is a very early version. As in, hacked together in a day and basically untested. So, please tell me if you encounter any bugs, and expect several updates before it works properly. Java is required for conversion (but not to run the converted applications).
